
# Todo Application  

A simple Todo Application made following MVVM Architecture with Jetpack Compose.

It is utilising:

    1. MVVM Architecture 
    2. Jetpack Compose
    3. Dagger Hilt - Dependency Injection
    4. Room - Local DB
